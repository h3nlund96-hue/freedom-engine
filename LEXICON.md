# Lexicon — Freedom Engine

The Freedom Engine world has its own language. Use these terms consistently across code, comments, UI text, and AI prompts. Avoid generic productivity vocabulary when a Freedom Engine term exists.

---

## The Founder

The Founder is Henrik in build mode. It is an identity and a role — not a page, module, or user account. When writing UI text or product copy, always address or refer to "The Founder."

---

## Freedom Engine

The long-term personal AI operating system being built. It is the larger world and system — not just an app. Freedom Engine contains modules, locations, Questlines, and Companions.

---

## AI Mastery HQ

The first module and starting headquarters inside Freedom Engine. It is where The Founder begins. Route: `/`

---

## Main Quest

The largest active mission. There is only one Main Quest at a time.

**Current Main Quest:** Build the Freedom Engine.

---

## Questline

A larger story path or project area inside Freedom Engine. Questlines contain multiple Quests.

**Current Questline:** AI Mastery HQ

---

## Quest

A meaningful goal inside a Questline. There is only one active Quest at a time.

**Current active Quest:** Quest 2 — Build Freedom Engine Alpha

---

## Build

A concrete action that moves a Quest forward. Builds are smaller and more specific than Quests. Every Quest is completed through Builds.

**Current Build:** Build #008 — Quest Board

> Every Quest is completed through Builds.

---

## Side Quest

A smaller, useful quest that does not belong to the active Questline. Side Quests support the main path without becoming the main path. They should not distract from the active Quest.

---

## Idea Vault

A safe archive for capturing ideas before they are ready to become Quests. The core rule: **Capture first, organize later.** Ideas sealed in the Vault do not interrupt the active Build.

Route: `/idea-vault`

---

## Quest Board

The strategic room where The Founder sees the full picture: Main Quest, active Questline, active Quest, current Build, Side Quests, and completed Builds.

Route: `/quest-board`

---

## Constitution Hall

The location containing The Founder Constitution. It is the moral compass of Freedom Engine — a sacred hall, not a settings page.

Route: `/constitution`

---

## The Founder Constitution

The principles that guide every major decision inside Freedom Engine:

1. Freedom First
2. Build
3. Curiosity
4. Tend the Fire

---

## Tend the Fire

A principle and daily reminder that small progress is enough. It is not a separate main location — it is embedded in AI Mastery HQ as a living reminder. The Founder only needs to add one ember to keep the adventure alive.

> One ember is enough.

---

## Companions

AI specialists inside Freedom Engine. They understand The Founder, the Constitution, and the current Quest. They are not generic chatbots. They have roles, personalities, and context.

*(Not yet implemented — future Questline)*

## Language Rules

| Use | Avoid |
|---|---|
| Build | Task, ticket, action item |
| Quest | Goal, objective, sprint |
| Questline | Project, epic, category |
| Location | Page, screen, view |
| Companion | Bot, assistant, agent |
| Seal in the Vault | Save, submit, add |
| Headquarters / HQ | Dashboard, home, hub |
| The Founder | User, you, member |
| Tend the Fire | Check in, motivate, stay on track |
| Side Quest | Minor task, nice to have |
| Enter | Click here, go to, open |
